- New **Integrations page** — browse available connections, see what's installed, and manage each one from a dedicated settings view
- **WhatsApp integration** is now available — connect your clinic's WhatsApp number to send appointment confirmations, reminders, and updates to patients automatically
- Manage your message templates, opt-out preferences, and daily send limits from the WhatsApp settings page
- Each integration shows its connection status and the last time it was active
