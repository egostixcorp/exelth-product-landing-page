- **Patient Flow overview** — see which patients are waiting, in-consultation, or done, all in one view
- **Task management** — create and assign internal tasks for your clinic team
- **Automated appointment reminders** — set up rules to automatically remind patients before their appointment via SMS or WhatsApp
- Reminders are created automatically when an appointment is booked, rescheduled, or cancelled
- Configure reminder timing, message content, and delivery channel from the Workflow settings
