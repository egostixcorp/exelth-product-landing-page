# Module Guide: Home Dashboard

The Home Dashboard is your clinic's central command center, offering a real-time overview of daily operations, setup checklist items, and quick action shortcuts.

---

## 1. Daily Greeting & Setup Checklist

Upon logging in, the dashboard greets you with your facility name and displays a progress tracker for configuring the system:

*   **Setup Progress Tracker (`GetStartedCards`):** Helps clinic owners and managers configure their digital workspace. It lists four key milestones:
    1.  *Facility Configured:* Complete the hospital profile details (address, contact details).
    2.  *Departments Created:* Register at least one operational specialty (e.g., General Medicine).
    3.  *Staff Ready:* Register practitioners and receptionist profiles.
    4.  *First Patient Visit Created:* Complete a patient registration and consultation cycle.

## 2. Quick Actions Shortcuts

Instantly perform frequent tasks from anywhere on the dashboard using the Quick Action shortcuts panel:

*   **Add Patient:** Opens the quick registration form.
*   **Book Appointment:** Opens the calendar scheduling panel.
*   **Block Doctor Dates:** Lets you mark unplanned leaves for doctors, temporarily stopping bookings for those dates.
*   **Check-in Patient:** Opens the reception desk queue manager.

## 3. Daily Schedules Calendar

The dashboard includes a simplified calendar view showing the day's scheduled appointments. This allows you to check upcoming patient slots, review assigned doctors, and view appointment status indicators without leaving the home screen.

---

## 4. Live Activity Log

The **Recent Activity** panel updates automatically to show real-time changes in your clinic:
*   Patient check-ins.
*   Consultation completions.
*   Uploaded laboratory test results.
*   Completed billing transactions.
