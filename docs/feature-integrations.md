# Module Guide: Third-Party Integrations

Exelth integrates with third-party tools to extend your clinic's capabilities.

---

## 1. Supported Integration Applications

The **Integrations** directory displays available connectors:

*   **WhatsApp (Active):** Connects your clinic's WhatsApp Business number to automate reminders, payment links, and test results.
*   **Excel (Planned):** Export data lists (e.g., patient directories, billing tables) to spreadsheets.
*   **Google Docs (Planned):** Save patient clinical files directly to a secure Google Drive folder.
*   **Slack (Planned):** Forward urgent alerts or roster changes to internal staff channels.
*   **Zapier (Planned):** Connect Exelth to 5000+ external apps.

---

## 2. Setting Up WhatsApp Business API

The WhatsApp integration is the primary patient communication channel:

1. Navigate to the **Integrations** → **WhatsApp** panel.
2. Link your WhatsApp Business account.
3. Configure message templates for automated notifications (e.g., confirmations, reminders, results).
4. Save and test the connection by sending a test message.
