# Platform Roadmap: Feature Updates & Release Timeline

This page outlines coming soon features and platform updates planned for Exelth.

---

## 1. Feature Release Timeline

| Release Target | Feature Module | Target Audience | Description |
|---|---|---|---|
| **Q3 2026** | **Excel Exports** | Clinic Managers | Download patient directories and revenue reports directly as Excel spreadsheets. |
| **Q3 2026** | **Google Docs Auto-Save** | Medical Staff | Automatically back up clinical consult notes and prescriptions to your secure Google Drive. |
| **Q4 2026** | **Smart Scheduling Agent** | Receptionists | Automated assistant that messages waitlisted patients when a slot opens up. |
| **Q4 2026** | **Automated Billing Agent** | Clinic Owners | Automatically compile invoices and send overdue payment reminders at the 48-hour mark. |
| **Q1 2027** | **Slack Alerts Channel** | Administrators | Forward urgent messages, schedule updates, or emergency leaves directly to your staff's Slack channel. |

---

## 2. In-Progress Quality Updates

We are constantly improving the platform experience:
*   **Vitals Tracking Graphs:** Adding visual history graphs for blood pressure and heart rate inside the EHR profile.
*   **Faster Messaging Alerts:** Upgrading the SMS and WhatsApp message delivery networks to reduce dispatch delays.
*   **Multi-Branch Roster Sync:** Simplifying shift handovers for doctors working across multiple clinic branches.
