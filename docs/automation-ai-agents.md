# Clinic Automation: AI Assistants & Bots

Exelth uses automated systems and AI assistants to handle clinic operations, follow-up messages, and patient communications.

---

## 1. Automated System Agents

Exelth uses background system agents to handle repetitive clinic tasks:

### 1. The Reminder Agent (Active)
*   **Role:** Manages the patient notification queue.
*   **Tasks:** Sends automated SMS and WhatsApp messages for appointment bookings, 24-hour reminders, follow-up alerts, and lab result ready notifications.

### 2. The Smart Scheduling Agent (Upcoming)
*   **Role:** Manages appointment queues and cancellations:
    *   **Waitlist Automations:** Messages waitlisted patients when a slot opens up.
    *   **Queue Balancing:** Alerts managers when a doctor is overloaded and helps reschedule appointments.

### 3. The Billing Agent (Upcoming)
*   **Role:** Manages invoice reminders:
    *   **Auto-Invoice:** Generates and sends invoices via SMS/WhatsApp upon consultation completion.
    *   **Payment Reminders:** Automatically alerts patients when payments are overdue.

---

## 2. ExelthAI Chat Assistant

The **ExelthAI** chat assistant is an AI tool available from the dashboard to help clinic staff check operational metrics:
*   Ask questions in plain English, such as: *"How many patients are waiting?"*, *"Who is the active doctor in Pediatrics?"*, or *"Show me the daily revenue from lab tests."*
*   The assistant securely retrieves the requested information and displays the results in the chat panel.
